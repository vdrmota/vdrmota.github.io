SPOTIFY PLAYLIST PREDICTIONS

Data Collection      EDA      Testing      Baseline      K-Means      Final: KNN      Conclusions, Future Work and Citations     

This is our CS109a final project on Spotify playlist predictions.


Problem Statement:

Exploring and finding new music is integral to keeping streaming services like Spotify afloat. The process of recommending new music to keep users on the platform requires ample research and uses immense amounts of past data, as well as machine learning algorithms. Our goal is to, given a user's input playlist, predict another playlist that the user is likely to enjoy. While interesting, this is a complicated task because it is tough to objectively define a well-predicted playlist. However, we attempt to do so using our innovative test metric.


Motivation:

Given the immensely subjective nature of the problem and the number of different approaches, in regards to Data Science techniques, can be taken to address this problem, we are motivated by the desire to experiment, learn and develop the knowledge and research in this area, while learning how to employ what we have learn throughout this class.
